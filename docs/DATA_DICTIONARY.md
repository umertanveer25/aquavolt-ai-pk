# 📖 AquaVolt-AI Data Dictionary

This document defines the schema and scientific methodology behind the telemetry data generated by **AquaVolt-AI**.

## Core Telemetry Variables

| Column | Description | Unit / Scale | Source / Methodology |
|---|---|---|---|
| `timestamp` | The exact UTC time the satellite data was captured or predicted. | ISO 8601 | Sentinel-2 / Landsat 8/9 / Open-Meteo |
| `latitude` / `longitude` | GPS Coordinates of the specific 10x10m pixel/sector. | Decimal Degrees | WGS84 (EPSG:4326) |
| `sector_row` / `sector_col` | The spatial grid index of the specific crop sector. | Integer | Local Field Grid |
| `field_name` | Name of the agricultural field. | String | Configuration |
| `scene_id` | The specific satellite scene/granule ID used for computation. | String | Sentinel Hub / EarthEngine |

## Vegetation & Optical Indices (Optical)

| Column | Description | Unit / Scale | Source / Methodology |
|---|---|---|---|
| `ndvi` | Normalized Difference Vegetation Index. Indicates plant health/vigor. | -1.0 to 1.0 | `(NIR - RED) / (NIR + RED)` |
| `ndwi` / `ndwi_real` | Normalized Difference Water Index. Plant water content. | -1.0 to 1.0 | `(GREEN - NIR) / (GREEN + NIR)` |
| `savi` | Soil Adjusted Vegetation Index. NDVI corrected for soil brightness. | -1.0 to 1.0 | `((NIR - RED) / (NIR + RED + L)) * (1+L)` |
| `lai` | Leaf Area Index. Density of plant canopy. | m²/m² | Derived from NDVI/SAVI |
| `fcover` | Fractional Vegetation Cover. Percentage of ground covered by crops. | 0.0 to 1.0 | `(NDVI - NDVI_soil) / (NDVI_veg - NDVI_soil)` |

## Thermal & Energy Balance Variables

| Column | Description | Unit / Scale | Source / Methodology |
|---|---|---|---|
| `lst` | Land Surface Temperature. | Celsius (°C) | Landsat 8/9 Thermal Band (TIRS) or MODIS |
| `lst_modis` | Land Surface Temperature fallback from MODIS. | Celsius (°C) | MODIS Aqua/Terra |
| `air_temp` | Ambient air temperature at 2m. | Celsius (°C) | Open-Meteo API |
| `humidity` | Relative Humidity at 2m. | % | Open-Meteo API |
| `solar_rad` | Incoming Solar Radiation. | W/m² | Open-Meteo API |
| `precip` | Precipitation (Rainfall). | mm | Open-Meteo API |
| `soil_temp` | Soil temperature at 0-10cm depth. | Celsius (°C) | Open-Meteo API |
| `soil_moisture` | Volumetric soil moisture at 0-10cm depth. | m³/m³ | Open-Meteo API |

## Hydrological & Irrigation (FAO-56 PM)

| Column | Description | Unit / Scale | Source / Methodology |
|---|---|---|---|
| `Kc` | Crop Coefficient. Derived dynamically from LAI and NDVI. | Dimensionless | `1.20 - exp(-0.8 * LAI)` |
| `Ks` | Water Stress Coefficient. Reduces ETc when soil water drops below RAW. | 0.0 to 1.0 | Derived from root zone depletion |
| `Dr` | Root Zone Depletion. Current water missing from the root zone. | mm | Water Balance Equation |
| `TAW` | Total Available Water in the root zone. | mm | Field Capacity - Permanent Wilting Point |
| `RAW` | Readily Available Water. Water crop can extract without stress. | mm | `TAW * p` (where p is depletion fraction) |
| `ETc` | Crop Evapotranspiration under standard conditions. | mm/day | `Kc * ET0` |
| `et0_deficit_7d` | Cumulative Evapotranspiration deficit over the last 7 days. | mm | Rolling sum of `ET0 - Precip` |
| `water_need` | **Core Target:** The exact volume of water needed to return soil to RAW. | mm | `Max(0, Dr - RAW)` |
